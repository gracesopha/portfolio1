AWS, DEVOPS, MARIADB, NODEJS, NGINX, PUG

M y senior capstone project for ITMT430 Systems Integration course. Using the comprehensive knowledge from all the classes taken previously, myself and my team built a three-tier web application called Hawkstagram - an Instagram-inspired app for IIT students. On the frontend, we utilized NodeJS, Pug and Nginx to bring a web-based application for students at IIT to post and view others' photos. As for the backend, we use MariaDB and AWS to store and retrieve our data. Additional features include Google single-sign-on API, and full automation of deployment using Vagrant, Promox and Terraform on an Ubuntu server.

Throughout this course, our team worked in three-week sprints over 16 weeks to accomplish a build of three web servers, one load balancer, and a database with a cloud-based architecture. We rotated roles of project manager, lead developer, UI/UX developer, and IT operations and systems manager to experience all types of labor in the value stream. It was a challenging, yet fulfilling semester applying all my studies! Please visit the repository below!
