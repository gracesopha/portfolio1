PYTHON & MYSQL

T he final project for my open-source programming class, created using Python and MySQL queries. This project takes public CSV data posted by the City of Chicago's Crime center to investigate items such as type of crime, date of crime, arrest status, location and so forth. With the collected data, using a simple GUI, the program displays visualizations of the analytics using a histogram, bar and pie chart for ease of depiction.
