All IT professionals will inevitably be forced on the receiving end of a trouble ticketing system. That is why for the final project of my Intermediate Software Development course, I programmed a help desk using Java and MySQL for both admins and standard users. By the type of user, they will receive one of two GUIs: admin and user. The admin has full CRUD (create, read, update, delete) implementations. This allows an admin to respond to and close ticket issues. A standard user is limited to creation of a ticket and the ability to read it.
