HTML, CSS, JAVASCRIPT

The final project for my Human-Computer Interactions course created in my team of four. This is a web development project built with HTML, CSS and JavaScript to seamlessly guide a user through a short (and accessible via ARIA) checkout process. The foundation of this project relies on user functionality and feedback, which was regularly tested throughout the development to improve the flow and succession of the webpage. With the firsthand knowledge and experience of UI/UX testing, our team developed "The Store Company" to (hypothetically) allow our users to shop amongst avaliable produce for delivery! Please visit the repository and GitHub-hosted page below to experience our simple checkout service!
